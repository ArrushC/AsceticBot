# Unique Bot
Unique Bot is a multi-purpose discord bot, designed to do everything uniquely as per as your needs.<br>Made by the Unique Bot Organistaion.



## Information:
- Made in Kotlin (a JVM langauge, thus similar to java)
- Discord API: Discord4J (D4J) with version `3.0.7`
- Bot version: `V1.0.0`
- Check the wiki for updates and information on the bot



> Currently this bot under development, henceforth features will be limited. Don't worry, it won't take long :)
